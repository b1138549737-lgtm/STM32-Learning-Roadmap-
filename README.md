# STM32 Learning Roadmap 🚀

> A Chinese student's hands-on STM32 MCU learning journey — from GPIO to PWM, ADC, timers, interrupts, and beyond.

这是我的 STM32 嵌入式学习记录，使用 **STM32F103** 系列单片机 + **Keil MDK v5** + **标准外设库**，从一个点灯开始逐步深入。

---

## 📂 项目列表（由浅入深）

### 第一阶段：GPIO 基础

| 项目 | 内容 | 关键知识点 |
|------|------|-----------|
| `1.1点灯` | LED 点亮 | GPIO 输出、`GPIO_SetBits/ResetBits` |
| `1.2流水灯` | 流水灯效果 | GPIO 多引脚操作、循环移位 |
| `1.3蜂鸣器` | 蜂鸣器控制 | GPIO 输出、PWM 基础 |
| `1.4按键控制小灯` | 按键控制 LED 亮灭 | GPIO 输入、按键消抖 |

### 第二阶段：传感器与显示

| 项目 | 内容 | 关键知识点 |
|------|------|-----------|
| `2.1光敏传感控制器` | 光敏传感器控制 LED | GPIO 数字输入、环境光检测 |
| `2.1控制显示屏` | OLED 显示屏驱动 + 光敏 + PWM 呼吸灯 | SPI/I2C 通信、ADC 模拟输入、PWM 输出、模块化封装 |

### 第三阶段：中断系统

| 项目 | 内容 | 关键知识点 |
|------|------|-----------|
| `3.1外部中断，光电编码器计次` | 外部中断计数 | EXTI、NVIC、中断优先级 |
| `3.2外部中断，旋转编码器计次` | 旋转编码器驱动 | 双边沿触发、正反转判断 |

### 第四阶段：定时器

| 项目 | 内容 | 关键知识点 |
|------|------|-----------|
| `4.1内部时钟，定时器定时中断` | TIM 定时中断 | TIM 时基、中断回调 |
| `4.1外部光电传感器模拟时钟` | 外部信号模拟时钟 | TIM 外部触发、捕获 |

### 第五阶段：PWM 与电机控制

| 项目 | 内容 | 关键知识点 |
|------|------|-----------|
| `5.1pwm控制呼吸灯` | PWM 呼吸灯效果 | TIM 输出比较、PWM 占空比调节 |
| `5.1pwm控制舵机` | 舵机角度控制 | 50Hz PWM、脉宽与角度映射（0.5ms~2.5ms） |
| `6-5 PWM驱动直流电机` | 直流电机调速 | PWM + 电机驱动、速度调节 |

### 第六阶段：输入捕获

| 项目 | 内容 | 关键知识点 |
|------|------|-----------|
| `6.1输入捕获测频率` | 测量信号频率 | TIM 输入捕获、频率计算 |
| `6.1输入捕获测占空比` | 测量 PWM 占空比 | 双通道捕获、脉宽与周期计算 |

### 工具

| 项目 | 内容 |
|------|------|
| `stm32工程模板` | 通用 STM32F103 工程模板，包含标准外设库 |

## 🛠 开发环境

| 工具 | 说明 |
|------|------|
| **MCU** | STM32F103C8T6 (Blue Pill) / STM32F103ZET6 |
| **IDE** | Keil MDK v5 (µVision) |
| **Library** | STM32 Standard Peripheral Library v3.5.0 |
| **调试器** | ST-Link / DAP-Link |

---

## 📌 代码规范

每个项目遵循一致的目录结构：

```
Project/
├── User/          ← 主程序 (main.c)
├── Hardware/      ← 硬件驱动 (OLED, Key, LED...)
├── BSP/           ← 板级支持包 (PWM, 传感器...)
├── System/        ← 系统文件 (Delay, SysTick)
├── Library/       ← 标准外设库（已预置）
└── Start/         ← 启动文件 + CMSIS
```

## 🎯 学习路线图

```
GPIO输出 → GPIO输入 + 消抖 → 外部中断 → 定时器中断
    → PWM输出 → 舵机/电机控制 → ADC采集
    → 输入捕获 → 模块化BSP封装 → 多外设协同
```
